# 市场全览与 RPS 强弱复盘模块 (Market Reviewer)

`market_reviewer.py` 是本系统的“数据体检仪”，它摒弃了外部不稳定的第三方指数，完全基于本地的日度连续 K 线数据，自下而上地为 70 个品种和 9 大板块生成真实的资金流向和价格强度排序。

## 1. 核心功能与方法

### A. 真实资金净流入量化
```python
def calculate_capital_flow(self, target_date: str = None) -> dict
```
- **描述**：量化全市场所有品种及板块在指定日期的资金净流向。
- **计算逻辑**：对于任何一个合约，其单日资金流入定义为 `(当日持仓量 - 昨日持仓量) × 结算价 × 合约乘数`。不区分双边/单边乘数，直接计算单边沉淀资金变动。
- **返回结构**：
  - `symbol_flow`: 单品种层面的资金流水降序表（帮你找出全市场谁在疯狂吸金，谁在遭遇抛售）。
  - `sector_flow`: 9大板块层面的总净流入聚合（帮你把握宏观板块轮动节奏）。

### B. 双模式自建板块指数合成
```python
def generate_sector_indices(self, start_date: str = '20200101') -> dict
```
- **描述**：从底层品种的价格数据出发，我们自行合成属于我们自定义的 9 大板块走势指数。
- **双模式**：
  - `nw_index` (持仓金额加权指数)：默认主力指数。用各品种 T-1 期的名义持仓金额占比作为权重，完美复刻大资金的宏观偏好（比如“原油油品”中，SC的权重将主导走势）。
  - `ew_index` (等权重指数)：辅助指数。忽略品种体量，所有成分股权重一致，专用来发现小品种的暴涨异动。

### C. 相对价格强度 (RPS) 横向评分
```python
def calculate_rps(self, periods: list = [20, 60, 120]) -> dict
```
- **描述**：寻找全市场的涨跌急先锋与滞后生。
- **机制**：
  - **单品种评分**：提取所有单品种的主力连续合约（`is_continuous=1`），计算过去 $N$ 天的收益率，并在 70 个品种内部进行排序打分（0-100）。得分越高代表趋势越强。
  - **板块评分**：基于上面合成的持仓加权板块指数，对 9 个板块进行宏观横向 RPS 打分，判定当前的风口板块。

### D. 板块内部龙虎榜
```python
def get_sector_leaders_laggards(self, symbol_rps_df: pd.DataFrame, period: int = 20) -> dict
```
- **描述**：如果你想在某个特定板块（如黑色建材）内部寻找交易标的，本函数将自动为你提取该板块内的“绝对龙头（最强品种）”和“龙尾（最弱品种）”。

## 2. 数据依赖与约束
- 高度依赖 `kline_daily` 表中的 `open_interest`（持仓量）和 `settlement`（结算价）字段。
- 所有 RPS 的品种评估均只认 `is_continuous = 1`（主连合约），避免了换月跳空对强弱判断的干扰。
- 该模块是前端复盘看板的底层引擎，所有计算均在内存的 Pandas DataFrame 中完成，运算极快。
